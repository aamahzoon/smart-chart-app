# 📊 Smart Chart Explorer

اپلیکیشن استریم‌لیت برای تشخیص خودکار مناسب‌ترین نمودار توصیفی از روی ساختار هر دیتاست (CSV / Excel)، به‌همراه امکان ساخت نمودار دلخواه.

## قابلیت‌ها
- آپلود فایل CSV یا Excel
- تشخیص خودکار نوع ستون‌ها (عددی، دسته‌ای، تاریخی، بولی)
- پیشنهاد و رسم خودکار نمودارهای مناسب (روند زمانی، هیستوگرام، پراکنش، میله‌ای، دایره‌ای، باکس‌پلات، ماتریس همبستگی)
- ساخت نمودار دلخواه با انتخاب نوع نمودار، محورها، رنگ‌بندی و تجمیع داده

## اجرای محلی
```bash
pip install -r requirements.txt
streamlit run app.py
```

## دیپلوی آنلاین
این پروژه برای اجرا روی [Streamlit Community Cloud](https://streamlit.io/cloud) آماده است. فایل ورودی: `app.py`
