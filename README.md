# 📊 Smart Chart Explorer

اپلیکیشن استریم‌لیت برای تشخیص خودکار مناسب‌ترین نمودار توصیفی از روی ساختار هر دیتاست (CSV / Excel)، به‌همراه امکان ساخت نمودار دلخواه.

## قابلیت‌ها
- آپلود فایل CSV یا Excel
- تشخیص خودکار نوع ستون‌ها (عددی، دسته‌ای، تاریخی، بولی)
- پیشنهاد و رسم خودکار نمودارهای مناسب (روند زمانی، هیستوگرام، پراکنش، میله‌ای، دایره‌ای، باکس‌پلات، ماتریس همبستگی)
- ساخت نمودار دلخواه با انتخاب نوع نمودار، محورها، رنگ‌بندی و تجمیع داده
- **تحلیل آماری تخصصی زیر هر نمودار** با scipy و statsmodels: رگرسیون OLS، ANOVA، آزمون نرمالیتی (Shapiro-Wilk)، تحلیل روند و ایستایی سری زمانی (ADF)، شاخص تمرکز و آزمون Chi-Square برای داده‌های دسته‌ای، و ماتریس همبستگی — همراه با بینش‌های تفسیری خودکار به فارسی
- **گزارش‌ساز / داشبورد اختصاصی:** امکان افزودن هر نمودار (به‌همراه تحلیل آن) به یک گزارش قابل چیدمان (تمام‌عرض/نصف‌عرض)، جابه‌جایی ترتیب، حذف آیتم‌ها، و خروجی گرفتن به‌صورت فایل HTML مستقل و قابل ارائه

## اجرای محلی
```bash
pip install -r requirements.txt
streamlit run app.py
```
